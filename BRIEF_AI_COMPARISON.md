### 核心差异

| 维度 | 规则系统 | AI 系统 |
|------|----------|---------|
| **生成方式** | 关键词匹配 + 模板拼接 | AI 理解事件 + 推理生成 |
| **标题风格** | "对象A、对象B：固定句式" | 自然语言，因果关系清晰 |
| **逻辑关系** | 生硬拼接，关系不清 | 明确的因果链条 |
| **核心论点** | 模糊或缺失 | 一句话说清楚要证明什么 |
| **为什么值得写** | 缺失 | 2-3句详细说明 |
| **证据需求** | 笼统 | 具体到数据类型和来源 |
| **写作指导** | 无 | 推荐框架和策略 |
| **可读性** | 难懂，需要大量修改 | 易懂，可直接使用 |

## 详细对比示例

### 示例 1：中东冲突选题

#### 旧系统输出

```
标题：外交部、伊朗：事件升级之外，长期秩序成本在怎样上升
核心命题：围绕外交部、伊朗，证明"事件升级之外，长期秩序成本在怎样上升"
```

**问题分析**：
1. "外交部" 和 "伊朗" 是两个不相关的实体，放在一起没有逻辑
2. "长期秩序成本" 是抽象概念，读者不知道指什么
3. 标题像是把关键词用顿号连接，然后加上固定句式
4. 没有说明为什么这个话题值得写

#### 新系统输出

```
标题：中东冲突如何重塑全球能源供应链

核心论点：当前市场关注油价短期波动，但真正的结构性变化是霍尔木兹海峡封锁风险正在重构全球能源供应秩序

为什么值得写：伊朗与美国的冲突升级不仅推高油价，更重要的是暴露了全球能源供应链对单一通道的依赖。这种结构性脆弱性被短期价格波动掩盖，但对长期投资决策和企业战略至关重要。当前正是理解这一转变的关键时点。

目标读者：宏观投资者、商品交易员、能源行业从业者、供应链管理者

证据需求：
- 霍尔木兹海峡的历史通行量数据和当前封锁风险评估
- 全球主要能源进口国的供应链依赖度
- 替代航线的成本和时间增量
- 历史上类似事件对供给秩序的影响

推荐框架：结构型
推荐策略：密度强化
```

**改进点**：
- ✅ 标题清晰：中东冲突 → 能源供应链重塑（明确因果）
- ✅ 论点具体：不是抽象的"秩序成本"，而是"霍尔木兹海峡封锁风险"
- ✅ 解释价值：说明为什么现在写这个话题重要
- ✅ 证据具体：列出需要什么数据，从哪里获取
- ✅ 写作指导：推荐用结构型框架，密度强化策略

### 示例 2：美联储政策选题

#### 旧系统输出

```
标题：美联储、中东这条线，市场是否低估了再通胀与利率预期回摆
核心命题：围绕美联储、中东，证明"市场是否低估了再通胀与利率预期回摆"
```

**问题分析**：
1. "美联储、中东这条线" - "这条线"是什么线？
2. "利率预期回摆" - 专业术语，但与前面的关系不清
3. 标题是疑问句，但没有给出方向
4. 读者看完标题不知道文章要说什么

#### 新系统输出

```
标题：中东冲突推高油价，美联储降息预期为何一再落空

核心论点：市场低估了中东冲突对再通胀的影响，油价上涨正在改变美联储的利率路径，降息预期需要重新定价

为什么值得写：市场年初押注美联储多次降息，但中东冲突引发的油价上涨正在改变这一预期。当前市场仍在用旧的通胀模型定价，但能源价格的结构性上涨可能迫使美联储维持高利率更长时间。这个预期差距为投资者提供了重要的交易机会。

目标读者：宏观投资者、债券交易员、基金经理、关注美联储政策的投资者

证据需求：
- 油价上涨对核心通胀的历史传导时滞和幅度
- 美联储最新会议纪要中对能源价格和通胀的表态
- 市场对美联储降息次数和时点的定价变化
- 历史上类似情景美联储的政策反应

推荐框架：热点解读型
推荐策略：角度发现
```

**改进点**：
- ✅ 因果链清晰：中东冲突 → 油价 → 通胀 → 美联储政策 → 降息预期落空
- ✅ 标题提出明确问题：为何降息预期落空
- ✅ 指出市场盲点：低估了再通胀影响
- ✅ 说明交易机会：预期差距可以利用

## 技术实现对比

### 旧系统（规则驱动）

```python
# 硬编码的 7 个冲突轴模板
CONFLICT_AXIS = {
    "policy_inflation": "市场是否低估了再通胀与利率预期回摆",
    "supply_order": "供给秩序重估会不会比价格波动更重要",
    "diplomacy_conflict": "事件升级之外，长期秩序成本在怎样上升",
    "risk_pricing": "风险溢价究竟在围绕什么变量重估",
    # ...
}

# 关键词匹配分类
def classify_variable(text):
    for var, keywords in VARIABLE_KEYWORDS.items():
        if any(kw in text for kw in keywords):
            return var
    return "market_sentiment"

# 模板拼接生成标题
def generate_title(objects, variable):
    conflict_axis = CONFLICT_AXIS[variable]
    if article_type == "结构稿":
        return f"{objects}：{conflict_axis}"
    elif article_type == "判断稿":
        return f"{objects}这条线，{conflict_axis}"
```

**局限性**：
- ❌ 只有 7 个固定模板，无法处理新话题
- ❌ 关键词匹配容易误判
- ❌ 字符串拼接导致标题生硬
- ❌ 没有理解事件的因果关系

### 新系统（AI 驱动）

```
AI 提示词核心原则：
1. 理解事件的因果关系和深层含义
2. 发现别人没看到的角度
3. 选择对读者有长期价值的话题
4. 用自然、流畅的语言表达

AI 任务：
1. 理解每个事件簇的核心内容和相互关系
2. 识别哪些话题值得深入写作
3. 为每个值得写的话题生成选题，包含：
   - 自然语言标题（15-30字）
   - 核心论点（一句话）
   - 为什么值得写（2-3句）
   - 目标读者
   - 证据需求和来源
   - 推荐的写作框架和策略
   - 5维评分
```

**优势**：
- ✅ AI 理解事件内容和因果关系
- ✅ 生成自然语言标题
- ✅ 推理为什么值得写
- ✅ 可以处理任何新话题
- ✅ 提供完整的写作指导
