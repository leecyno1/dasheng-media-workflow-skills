## 💡 Tips

### For Efficient Execution
1. Set up browser window to 1920x1080 before starting
2. Open all screenshot URLs in tabs first
3. Use Cmd+Shift+4 for quick screenshots on macOS
4. Download images in parallel while taking screenshots
5. Create quality reports as you go (don't batch at end)

### For Quality Assurance
1. Always verify resolution with `file` command
2. Check file size >500KB for quality
3. Ensure text is readable at 100% zoom
4. Confirm source branding is visible
5. Document any deviations in quality reports

### For Troubleshooting
1. If URL blocked → Try incognito mode
2. If image too small → Look for "original size" option
3. If content changed → Screenshot what's available, note in report
4. If source unavailable → Use alternatives in MATERIAL_DOWNLOAD_INSTRUCTIONS.md
