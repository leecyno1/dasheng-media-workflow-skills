# baoyu-skills 内容创作技能包推荐
	
推荐一个宝玉老师的内容创作Skills仓库：jimliu/baoyu-skills，主要有下面几个skills：
	
| Skill | 功能 |
|-------|------|
| baoyu-slide-deck | 幻灯片生成 |
| baoyu-article-illustrator | 文章配图 |
| baoyu-cover-image | 封面图生成 |
| baoyu-xhs-images | 小红书图片 |
| baoyu-comic | 漫画生成 |
| baoyu-post-to-wechat | 发布到微信 |
| baoyu-post-to-x | 发布到X |
| baoyu-infographic | 信息图生成 |
	
这些Skills对中文用户特别友好——小红书图片、发微信、文章配图，都是我们平时会用到的。
	
安装命令：
```bash
npx skills add jimliu/baoyu-skills --yes
```
