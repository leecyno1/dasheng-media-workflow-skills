## 核心问题

用户反馈旧系统生成的选题"题目特别硬，逻辑关系像生搬硬套，没有特别强的逻辑关系和背后的逻辑意义"。

### 问题根源

旧系统使用硬编码模板拼接：
```python
# 7 个固定的冲突轴模板
conflict_axis = {
    "policy_inflation": "市场是否低估了再通胀与利率预期回摆",
    "supply_order": "供给秩序重估会不会比价格波动更重要",
    "diplomacy_conflict": "事件升级之外，长期秩序成本在怎样上升",
    # ...
}

# 标题 = 对象 + 连接词 + 固定冲突轴
title = f"{object_phrase}：{conflict_axis}"
# 或
title = f"{object_phrase}这条线，{conflict_axis}"
```

## 对比结果

### 选题 1：中东冲突

**旧系统（模板拼接）**：
```
标题：外交部、伊朗：事件升级之外，长期秩序成本在怎样上升
问题：
- "外交部" 和 "伊朗" 之间没有明确逻辑关系
- "长期秩序成本" 是抽象概念，与具体事件脱节
- 读者看不懂在说什么
```

**新系统（AI 推理）**：
```
标题：中东冲突如何重塑全球能源供应链
核心论点：当前市场关注油价短期波动，但真正的结构性变化是霍尔木兹海峡封锁风险正在重构全球能源供应秩序
为什么值得写：伊朗与美国的冲突升级不仅推高油价，更重要的是暴露了全球能源供应链对单一通道的依赖...
目标读者：宏观投资者、商品交易员、能源行业从业者、供应链管理者
推荐框架：结构型
推荐策略：密度强化
```

**改进点**：
- ✅ 标题自然流畅，清晰表达因果关系
- ✅ 核心论点明确，有具体判断
- ✅ 解释了为什么现在值得写
- ✅ 明确了目标读者和证据需求

### 选题 2：美联储政策

**旧系统（模板拼接）**：
```
标题：美联储、中东这条线，市场是否低估了再通胀与利率预期回摆
问题：
- "美联储" 和 "中东" 的关系不清
- "这条线" 是什么线？
- "利率预期回摆" 与前面的关系不明
```

**新系统（AI 推理）**：
```
标题：中东冲突推高油价，美联储降息预期为何一再落空
核心论点：市场低估了中东冲突对再通胀的影响，油价上涨正在改变美联储的利率路径，降息预期需要重新定价
为什么值得写：市场年初押注美联储多次降息，但中东冲突引发的油价上涨正在改变这一预期...
目标读者：宏观投资者、债券交易员、基金经理
推荐框架：热点解读型
推荐策略：角度发现
```

**改进点**：
- ✅ 清晰的因果链：中东冲突 → 油价 → 通胀 → 美联储政策
- ✅ 标题提出明确问题：为何降息预期落空
- ✅ 指出市场的认知盲点（低估再通胀）

```
推荐框架：纯观点型
推荐策略：角度发现
```

**改进点**：
