# Rewrite Channel Packs

本目录保存 `rewrite` 环节的渠道配置。

当前支持：

- `wechat_article.yaml`
- `wechat_video.yaml`
- `xiaohongshu_note.yaml`
- `xiaohongshu_video.yaml`
- `douyin_video.yaml`
- `bilibili_video.yaml`
- `video_account_video.yaml`
- `weibo_note.yaml`
- `weibo_video.yaml`
- `x_note.yaml`
- `x_video.yaml`

说明：

- 微信、小红书、微博、X 都显式区分图文/文字与视频
- 其他平台默认以视频包为主
