# Rewrite DNA 档案

本目录保存 `rewrite` 环节可直接调用的 DNA 卡。

每个文件对应一种文风 DNA，建议字段：

- `dna_id`
- `display_name`
- `opening_mode`
- `argument_style`
- `sentence_profile`
- `paragraph_profile`
- `punctuation_profile`
- `emotion_baseline`
- `explicit_judgment`
- `irony_level`
- `cta_style`
