# 07 发布 Prompt

## 阶段目标

围绕同一选题的已确认改写版本与补素材包，形成最终发布包并安排发布顺序。

## 适用输入

- 指定渠道的最终改写稿
- 标题池
- 摘要
- 图表/图片链接
- 补素材回填说明
- 渠道偏好

## 输出要求

1. 发布主包
2. 各渠道发布成品
3. 发布时间计划
4. 发布检查清单
5. 人工干预位

## 完成检查

- 是否完成四类渠道成品
- 是否体现渠道差异而不是同文复用
- 是否给出发布时间建议
- 是否为人工发布前复核预留了明确入口
