# 平台路由规则

本文档定义 distribute 阶段中，改写变体（variant）与目标平台的映射关系。

| 平台 | 主选变体 | 备选变体 | 是否需要视频 |

## 多选题（多 Topic）的路由逻辑

当 run_id 包含多个 topic 时：

### 微信公众号
- **每个 topic 各发一篇**：每篇独立草稿，不合并
- 两篇建议错开发布时间（当天和次日，或早晚各一篇）
- 默认选 `wechat_luxun_hot`，若用户指定则用指定变体

### 微信视频号
- **每个 topic 各发一条视频**，用对应视频文件
- 使用 `wechat_channels` 变体中的元数据（标题、描述、话题）
- 视频规格：支持横版和竖版，≤30 分钟

### B站
- **每个 topic 各发一条视频**，用对应视频文件
- 使用 `bilibili` 变体中的元数据（标题、简介、标签、分区）
- 视频规格：横版 16:9 或竖版 9:16
- 两条之间建议间隔 ≥1 小时

### 小红书
- **每个 topic 各发一条视频**，用对应视频文件
- 两条之间必须间隔 ≥5 分钟（平台限制）
- 可选：同一 topic 发两个版本（一天发一个）

### 抖音
- **每个 topic 对应一个视频文件**，依次发布
- 若 Stage 6 只生成了 narrative 视频，无 interactive chart 视频，则只发 narrative 版
- 发布间隔参考平台建议（≥30 分钟）

### 微博
- **从 rewrite_manifest.json 读取微博摘要元数据**
- 格式：`今天两篇分析：\n1. {topic_1_title}\n2. {topic_2_title}\n{共用CTA}`
- 可选：分开发两条，但间隔 ≥2 小时

### X (Twitter)
- **从 rewrite_manifest.json 读取推文串元数据**
- 若话题相关性强（如同属 AI 领域），合并为串；若话题差异大，分开发

## 变体选择决策树

```
用户有明确指定？
  ├─ 是 → 用指定变体
  └─ 否 → 使用默认路由（见上表"主选变体"）

目标受众是什么？
  ├─ 需要强观点/财经类读者 → luxun 版
  └─ 需要普通读者/情感共鸣 → lemon 版

是否有视频资源？
  ├─ 有 Stage 6 视频 → 小红书/抖音完整执行
  └─ 无视频 → 小红书/抖音跳过，记录 pending
```

## 封面图路由

封面图来源优先级：

1. Stage 4 Material 中已下载的图片（`产物/04_Material/{run_id}/{topic}/images/`）
2. Stage 6 视频第一帧截图（自动提取）
3. 无图时：平台 skill 的默认封面逻辑（wechat-multi-publisher 会自动从 Unsplash 搜索）

## 不同平台的字数/时长规格

| 平台 | 文字内容规格 | 图片规格 | 视频规格 |
|------|------------|---------|---------|
| 微信公众号 | 无上限，建议 2000-5000 字 | 封面 900x500px | 嵌入式，≤20MB |
| 微信视频号 | 标题 ≤30字，描述 ≤1000字 | 自动提取或手动上传 | 横版/竖版，≤30分钟 |
| 小红书视频 | 标题 ≤20字，正文 ≤1000字 | 1080x1440px（竖版） | 竖版 9:16，≤15分钟 |
| 抖音 | 字幕 ≤15字/段，描述 ≤1000字 | 封面 1080x1920px | 竖版 9:16，≤15分钟 |
| B站 | 标题 ≤80字，简介 ≤2000字 | 封面 1146x717px（16:9） | 横版/竖版，≤4小时 |
| 微博 | ≤2000字（含链接），摘要建议 ≤140字 | 最多9张，建议1张 | ≤15分钟 |
| X (Twitter) | 主推文 ≤280字，推文串不限数量 | 最多4张 | ≤2分20秒 |

## 话题标签策略

**注意**：所有平台的话题标签现在从 `rewrite_manifest.json` 的 `platform_metadata` 中读取，不再临时生成。

### 小红书
每篇视频配 5-8 个话题标签，格式 `#话题名`：
- 大话题（百万级）×2：`#财经` `#投资` `#AI`
- 中话题（10-100万）×3：`#AI投资` `#科技趋势`
- 小话题（精准）×2：`#{topic_keyword}` `#选题相关词`

### 抖音
配 1-3 个话题标签

### B站
配 ≤10 个标签，每个 ≤20 字

### 微信视频号
配 1-3 个话题，格式 `#话题名`

### 微博
配 1-2 个话题标签，格式 `#话题名#`：
- 选热点话题，优先用微博热搜相关词

### X
配 2-3 个 hashtag，格式 `#Tag`：
- 英文 hashtag 优先（国际受众）
- 示例：`#AIinvestment` `#China15thFiveYearPlan` `#Tech`
