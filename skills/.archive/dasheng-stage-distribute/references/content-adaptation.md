# 内容适配指南

本文档说明如何将改写稿适配为各平台所需的格式。

---

## 微博摘要生成（≤140字）

**来源**：wechat_luxun_hot.md 或 wechat_lemon_normal.md

**生成规则**：

```
结构：[反直觉论点] + [核心数据1个] + [结论句] + [话题标签]

示例：
"全球媒体都说中国经济放缓，但我看到的是另一件事——
4.5%的背后，国务院发展研究中心数据显示6%完全可行，但主动选了不做。
这不是失速，是换挡。#十五五规划#"

字数控制：
- 核心句 ≤80字
- 话题标签 ≤20字（1-2个）
- 总计 ≤140字
```

**质量检查**：
- [ ] 包含1个具体数据或名词
- [ ] 有反直觉或悬念感
- [ ] 不直接复制文章标题
- [ ] 结尾有明确的立场或结论

## X 推文串生成

**来源**：两个 topic 的 wechat_luxun_hot.md

**推文串结构**（每条 ≤280字，建议中文 ≤130字）：

```
主推文（Hook）：
└─ 最反直觉的核心论点，提问式或数据式开头

1/n 数据支撑：
└─ 最有力的1个数据点 + 为什么重要

2/n 影响层面：
└─ 对读者的直接影响（职场/投资/行业）

3/n 总结与CTA：
└─ 一句话结论 + 欢迎关注/评论

可选：英文摘要推文（单独发，不作为串的一部分）
```

**示例**（参考 Topic 2）：
```
主推文：
中国把GDP目标降到4.5%，全球媒体都说是"放缓"。
但一个被忽视的数据：国务院发展研究中心说6%完全可实现，他们主动选了不做。
这背后是什么逻辑？🧵

1/4：
地方政府债务率已达GDP 120%（IMF数据，警戒线是60%）。
继续用旧刺激方式跑6%+，短期好看，长期是债务危机的燃料。

2/4：
所以十五五选了三条新路：AI贡献GDP从3%→15%、消费占比从38%→45%、关键技术自给率从60%→80%。
这不是"做不到"，是"不想那样做"。

3/4：
对普通人来说：AI工具进入工作流的速度会加快，传统地产和低端出口制造的刺激不会再来。
转型比等待现实。
#十五五规划 #ChinaEconomy
```

## 抖音字幕脚本格式化

**来源**：xhs_video_luxun_hot.md 或 xhs_video_lemon_normal.md

**格式规则**：

```
输入：视频脚本（分段口播文本）
输出：抖音字幕格式（每段 ≤15字，标注粗略时间轴）

转换规则：
1. 每个"句子"（句号/问号结束）作为一个字幕块
2. 超过15字的句子，在语义停顿处断开
3. 强调词/数字做标注：【120%】【4.5%】
4. 删除脚本中的[括号标注]、制作备注等非口播内容

示例输出：
[00:00] 今天聊一个
[00:02] 全球媒体都没说透的事
[00:04] 2026年中国GDP目标
[00:07] 【4.5%-5%】
[00:09] 二十年来最低
[00:12] 彭博社说：放缓
[00:15] 我说：他们看错了
```

**字幕文件格式**（输出为 SRT 兼容格式）：
```
1
00:00:00,000 --> 00:00:02,000
今天聊一个

2
00:00:02,000 --> 00:00:05,000
全球媒体都没说透的事

...
```

## 小红书封面文案

**来源**：xhs_video 脚本的开场部分

**规格**：
- 封面标题：≤20字，有悬念或数字
- 封面副文案：≤15字，补充说明

**生成规则**：
```
从视频脚本开场 Hook 中提取最强的一句话
压缩到 ≤20字
加入数字强化：【1.2万亿】【4.5%】【全球媒体误读了】

示例：
封面标题："全球媒体误读了中国的4.5%！"（15字）
封面副文案："国务院数据告诉你真相"（12字）
```

## 平台差异化处理原则

### 语调调整

| 平台 | 语调 | 语速感 | 亲密感 |
|------|------|--------|--------|
| 微信 | 深度分析，可以复杂 | 慢读 | 中等 |
| 小红书 | 轻松直白，像朋友 | 快节奏 | 高 |
| 抖音 | 口语化，有节奏感 | 很快 | 高 |
| 微博 | 简短有力，带观点 | 中等 | 中 |
| X | 精炼，可英文 | 快 | 低 |

### 数据使用

- **微信/深度内容**：完整引用来源（"国务院发展研究中心报告显示……"）
- **微博/X**：保留数字，省略来源（"地方债务率已达120%"）
- **小红书/抖音**：数字视觉化表达（"地方政府欠了相当于GDP 1.2倍的债"）

### emoji 使用策略

- **微信**：不用或极少用（保持专业感）
- **小红书**：适量使用相关 emoji（📊💡🔑）
- **抖音**：字幕不用，但口播停顿可用节奏词
- **微博**：1-2个功能性 emoji 即可
- **X**：可用表情辅助阅读（🧵表示推文串，📊表示数据）
