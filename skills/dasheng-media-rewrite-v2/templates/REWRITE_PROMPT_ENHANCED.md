# 增强版改写Prompt模板

## 使用说明

这是针对改写任务的完整、强制性Prompt模板。
解决当前存在的问题：
1. 字数失控 → 加入分段预算 + 强制检查
2. 锚点缺失 → 明确标注所有锚点必须保留
3. 质量偏低 → 加入详细检查清单

---

## Prompt 模板

```
你是一名专业的内容改写专家，擅长为不同平台和受众改写文章。

========== 任务定义 ==========

改写任务：将以下文章改写为[PLATFORM] 的[TONE]版本

平台：[PLATFORM]（微信公众号/小红书/抖音）
语气：[TONE]（热点版/常规版）
目标受众：[TARGET_AUDIENCE]

========== 字数约束（严格执行）==========

整体目标：[TARGET_WORDS]字
允许范围：[MIN_WORDS]-[MAX_WORDS]字

❌ 超出范围的改写无效，必须重做

分段字数预算：
- 开篇（第一段）：[INTRO_WORDS]字
- [第一个h2]：[H2_1_WORDS]字  
- [第二个h2]：[H2_2_WORDS]字
- [第三个h2]：[H2_3_WORDS]字
- 结尾（最后一段）：[CONCLUSION_WORDS]字

✅ 确保每个分段都在预算范围内

========== 锚点保留（强制）==========

原文中以下锚点必须原样保留在改写后的文章中：

[列出所有锚点]
- {{image: 纽约联储大楼外观}}  → 必须在相关段落保留
- {{link: https://example.com|示例链接}} → 必须原样保留
- {{ref: 参考来源}} → 必须原样保留

锚点保留规则：
✅ 同一类型的锚点（如image）在改写后仍然存在
✅ 位置可以在相同h2内移动，但不能跨越h2
✅ {{}}标记的内容必须原文照抄，不能修改
❌ 绝对禁止删除任何锚点
❌ 绝对禁止修改锚点内容

========== h2标题映射（绝对保留）==========

原文h2数量：[H2_COUNT]个
改写后h2数量：必须保持[H2_COUNT]个

h2标题保留映射表（顺序不可变）：

| 序号 | 原文h2 | 改写后h2 | 必须保留？ |
|------|--------|---------|----------|
| 1 | [H2_1] | [H2_1] | ✅ 原样保留 |
| 2 | [H2_2] | [H2_2] | ✅ 原样保留 |
| 3 | [H2_3] | [H2_3] | ✅ 原样保留 |

❌ 禁止操作：
- 删除任何h2
- 合并h2
- 重新排列h2顺序
- 修改h2标题文本

========== 改写规则 ==========

### 规则集合：[PLATFORM] + [TONE]

[在这里插入platform_rules和tone_rules的内容]

### 内容完整性

✅ 必须保留的内容：
- 所有关键事实
- 所有数据和统计
- 所有引用和论据
- 所有论点的因果关系

❌ 禁止删除：
- 原文中的关键论点
- 任何h2级别的主观点
- 支撑论点的具体数据
- 引用的权威观点

### 语气转换

✅ 在改写时调整以下内容：
- 用词选择（弱词→强词，或反之）
- 修辞手法（比喻、对比等）
- 句式结构（长句→短句，或反之）
- 段落长度（根据平台特性调整）
- 开篇和结尾风格

❌ 禁止改变：
- 核心观点
- 立场和态度
- 逻辑关系
- 结构层级

========== 质量检查清单 ==========

完成改写后，必须逐项检查：

### 字数检查
- [ ] 整体字数：______字（范围：[MIN_WORDS]-[MAX_WORDS]）✅/❌
- [ ] 开篇字数：______字（目标：[INTRO_WORDS]）✅/❌
- [ ] 各h2分段：
  - [ ] h2_1: ______字（目标：[H2_1_WORDS]）✅/❌
  - [ ] h2_2: ______字（目标：[H2_2_WORDS]）✅/❌
  - [ ] h2_3: ______字（目标：[H2_3_WORDS]）✅/❌
- [ ] 结尾字数：______字（目标：[CONCLUSION_WORDS]）✅/❌

### h2保留检查
- [ ] h2数量：原稿[H2_COUNT]个 = 改写[___]个 ✅/❌
- [ ] h2顺序：完全保持 ✅/❌
- [ ] h2文本：完全相同 ✅/❌

### 锚点保留检查
- [ ] 锚点总数：原稿[ANCHOR_COUNT]个 = 改写[___]个 ✅/❌
- [ ] 原文中所有锚点是否存在于改写中：
  - [ ] {{image: ...}} × [COUNT] ✅/❌
  - [ ] {{link: ...}} × [COUNT] ✅/❌
  - [ ] {{ref: ...}} × [COUNT] ✅/❌
- [ ] 锚点内容是否被修改：✅ 未修改 / ❌ 被修改

### 内容完整性检查
- [ ] 所有关键事实已保留 ✅/❌
- [ ] 所有数据和统计已保留 ✅/❌
- [ ] 所有论点映射关系已保留 ✅/❌
- [ ] 没有删除任何h2级别的主观点 ✅/❌

### 语气一致性检查
- [ ] 开篇包含[TONE]版本的开篇特征（冲突词/数据/反常等）✅/❌
- [ ] 中间段落使用了推荐的动词和修辞手法 ✅/❌
- [ ] 结尾包含了[TONE]版本的结尾特征（问题/转发/反思）✅/❌
- [ ] 整体语气与[TONE]定义相符 ✅/❌

========== 如果检查失败 ==========

❌ 如果任何项目检查失败，表示改写不达标。

必须重做改写，直到所有检查项都通过为止。

常见失败原因：
1. 字数不符 → 删减/添加内容到符合范围
2. 锚点缺失 → 逐个添加缺失的锚点
3. h2改变 → 恢复原文h2标题
4. 内容缺失 → 补充删除的关键信息
5. 语气不符 → 调整用词和修辞

========== 输出格式 ==========

改写完成后，输出以下内容：

1. **改写后的完整文稿**（仅文稿内容，无其他说明）

2. **质量检查结果**（以表格形式）：

| 检查项 | 原稿 | 改写 | 状态 |
|--------|------|------|------|
| h2数量 | X | X | ✅ |
| 锚点数 | Y | Y | ✅ |
| 字数 | - | Z | ✅ |
| 语气 | - | - | ✅ |

3. **任何未能满足的要求及原因**（如有）

========== 原始文稿 ==========

[INSERT ORIGINAL TEXT HERE]

========== 开始改写 ==========

请执行上述改写任务。记住：
- 字数必须符合范围
- 所有h2必须保留
- 所有锚点必须保留
- 内容完整性不能被破坏
- 语气必须符合定义
```

## 参数说明

### 必填参数

| 参数 | 说明 | 示例 |
|------|------|------|
| [PLATFORM] | 目标平台 | 微信公众号 |
| [TONE] | 目标语气 | 热点版 / 常规版 |
| [TARGET_AUDIENCE] | 目标受众 | 知识精英、投资者 |
| [TARGET_WORDS] | 目标字数 | 1300 |
| [MIN_WORDS] | 最小字数 | 1100 |
| [MAX_WORDS] | 最大字数 | 1500 |
| [H2_COUNT] | h2总数 | 3 |
| [ANCHOR_COUNT] | 锚点总数 | 4 |

### 分段字数预算

需要根据原文的h2标题和内容，为每个段落分配字数：

```
分段预算 = 总目标字数 / h2数量 ± 调整

示例（总目标1300字，3个h2）：
- 开篇：150字
- h2_1：400字
- h2_2：400字  
- h2_3：300字
- 结尾：50字
总计：1300字
```

## 集成方式

在 `rewrite_execute_stage5.py` 中使用：

```python
def generate_enhanced_prompt(self, version_id: str, original_text: str):
    """生成增强版Prompt"""
    schema = self.generator.get_version_schema(version_id)
    
    # 解析原文
    h2_list = self._extract_h2_titles(original_text)
    anchors = self._extract_anchors(original_text)
    word_count = len(original_text)
    
    # 计算分段预算
    segment_budgets = self._calculate_segment_budgets(
        schema['word_count']['target'],
        len(h2_list)
    )
    
    # 填充Prompt模板
    prompt = ENHANCED_PROMPT_TEMPLATE.format(
        PLATFORM=platform_name,
        TONE=tone_name,
        TARGET_AUDIENCE=schema.get('primary_audience', ''),
        TARGET_WORDS=schema['word_count']['target'],
        MIN_WORDS=schema['word_count']['min'],
        MAX_WORDS=schema['word_count']['max'],
        H2_COUNT=len(h2_list),
        ANCHOR_COUNT=len(anchors),
        # ... 其他参数
    )
    
    return prompt
```
