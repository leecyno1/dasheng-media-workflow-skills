# 小红书(XHS) 平台特异化规则

## 平台特征
- 目标受众：年轻、追求实用、视觉导向
- 阅读场景：快速浏览、视觉驱动
- 内容形式：标题 + 简短正文 + 配图重要
- 传播机制：赞、收藏、分享、评论

## 字数规范

| 版本 | 目标字数 | 建议范围 |
|------|--------|----------|
| Hot | 900 | 750-1050 |
| Normal | 650 | 500-800 |

## 写作风格要求

### 标题设计（极其重要）

**Hot版本标题**:
```
✅ "AI vs 华尔街：新模型引发紧急会议，金融业面临重塑"
✅ "稳定币牌照落地！传统巨头纷纷入场，加密市场迎新格局"
✅ "日本自卫队混入极端分子？系统性渗透曝光全过程"
```
特征：
- 字数20-35字（XHS标题极限）
- 包含数字或对比
- 包含感叹号或问号
- 使用视觉词汇（"曝光", "重塑", "纷纷入场"）

**Normal版本标题**:
```
✅ "AI新模型对金融业的三大风险"
✅ "稳定币牌照：监管新时代"
✅ "日本自卫队事件深度分析"
```
特征：
- 字数15-25字
- 直接切入主题
- 避免标点符号

### 正文结构（关键：上稀下密）

```
第1段（hook）：30-50字
  - 开门见山
  - 包含1个"为什么你应该读此文"
  - {{image: 配图1}}

第2段（背景）：80-100字
  - 简要事件背景
  - 为什么重要

第3段（核心观点1）：100-120字
  - 1个观点 + 2-3个数据或论据
  - {{image: 配图2}}

第4段（核心观点2）：100-120字
  - 与观点1对比或补充
  - 可包含实际案例

第5段（结论）：50-80字
  - 行动建议或反思
  - Hot版本：包含"点赞"激励
```

### 视觉词汇密度（重要）

**Hot版本**:
- 动词优先："暴雷", "内卷", "逆袭", "翻盘", "入局"
- 修饰词："意想不到", "出乎意料", "违反直觉"
- 数字：每段至少1个数据点
- 对比："曾经...如今..."

**Normal版本**:
- 使用相对温和的词汇
- 数据密度较低
- 学术性词汇可适度增加

### Emoji使用规范

**Hot版本**:
- 总数：4-6个
- 位置：分散在各段落，每段最多1个
- 禁用：❤️ 👍 💯（显得低龄化）
- 推荐：📊 💡 ⚡ 🔥 🎯 📈

**Normal版本**:
- 总数：0-2个（可选）
- 位置：标题或结尾

## 排版规范

### XHS特殊格式

```
【XHS标题】
粗体关键词用粗体

正文段落

🔑 关键点1
🔑 关键点2  ← Hot版本使用emoji加分点

结论段落

#标签1 #标签2 #标签3  ← Hot版本必须
```

### 配图锚点
- ✅ 必须保留所有{{image:}}锚点
- ✅ 建议在段落之间插入（不在段落中间）
- ✅ Hot版本至少3张图，Normal版本至少2张

## 常见错误与修正

| 问题 | 原文 | 改正后 |
|------|------|--------|
| 标题太长 | "人工智能新模型在金融市场应用的风险分析" | "AI新模型引发金融业紧急会议" |
| 文风不符 | "此现象系统性风险演化机制..." | "这个风险有多严重？💡 4大案例告诉你" |
| 词汇低龄化 | 过多emoji（😊😍🥰） | 改用专业emoji（📊⚡🎯） |
| 段落过长 | 单段300字 | 拆分为150字/段 |
| 信息密度低 | 用词重复、观点冗余 | 精简到80-120字/核心观点 |
| 锚点丢失 | 删除配图描述 | 保留{{image:}}在相关位置 |

## 质量检查清单

**标题**:
- [ ] Hot版本包含惊叹号或感叹
- [ ] 标题20-35字（Hot）或15-25字（Normal）
- [ ] 包含数据或对比

**正文**:
- [ ] 字数符合（Hot: 750-1050，Normal: 500-800）
- [ ] h2标题完全保留
- [ ] 所有{{image:}}锚点保留
- [ ] 段落长度80-150字（XHS偏好短段）
- [ ] Hot版本emoji 4-6个，均匀分布
- [ ] 视觉词汇密度适当

**参与度**:
- [ ] Hot版本包含"点赞"或"转发"激励
- [ ] 结尾包含反思问题（可选）
- [ ] 标签数3-5个（Hot版本）
