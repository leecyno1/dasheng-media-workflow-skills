# Normal版本（常规专业）语气规则

## 核心特征
- **目标**: 信息传递准确、逻辑严密、建立信任
- **受众心理**: 理性分析、求知、决策参考
- **关键词**: 深度、平衡、证据、权衡、结论

## 语气框架

### 开篇策略（清晰定位）

**结构：问题 + 为什么重要 + 本文切角**

```
示例1:
"稳定币牌照落地后，传统金融机构如何布局？
这个问题不仅关系到加密市场的未来，
更涉及整个金融体系的数字化转型。
本文从3个维度深度分析..."

示例2:
"日本自卫队事件引发关注。
表面上是个别违纪行为，
但背后反映了什么系统性问题？
我们通过数据和案例来拆解..."
```

**开篇原则**:
- ✅ 直接切入主题（无冗余）
- ✅ 解释为什么值得读（意义感）
- ✅ 承诺本文的分析角度（期待管理）
- ❌ 无夸张修辞，无感叹号
- ❌ 无"让我们来看"这类老气表述

### 中间段落策略（论证充分）

**段落结构（SPA）**:
```
S (Statement): 核心观点 
  "稳定币牌照的门槛建立了市场准入制度"

P (Proof): 论据支撑
  - 数据："资本充足率、审计要求..."
  - 案例："A国的经验是..."
  - 权威："监管机构表示..."
  - 反面："过去XXX导致..."

A (Application): 意义应用
  "这意味着：机构入场、市场洗牌、投资机会"
```

**段落长度**: 200-250字（适合深度阅读）

**论据均衡**:
- 不低于2个具体支撑
- 至少1个来自权威/数据
- 至少1个来自实例/反面
- 避免全是理论、全是数据、全是故事

### 结尾策略（建立认知）

**结构：总结 + 隐含意义 + 前景思考**

```
示例1:
"总体来看，稳定币牌照代表了监管的介入，
这标志着加密市场从'野蛮生长'向'制度建设'的转变。
这对投资者的启示是：规则制定者往往是最大赢家。
未来应关注政策导向而非短期价格波动。"

示例2:
"日本自卫队事件本质上反映了右翼渗透的系统性风险。
这不仅是日本的问题，也给其他国家敲响了警钟：
制度防护必须从源头开始，而非被动应对。
观察日本的后续处置，或可预测全球应对的方向。"
```

**结尾原则**:
- ✅ 不引入新信息（只总结已有观点）
- ✅ 层级递进（具体→抽象→前景）
- ✅ 包含"隐含意义"（for whom, so what）
- ❌ 无转发激励、无emoji
- ❌ 无夸大表述

## 词汇库

### 观点表述（客观但有立场）
```
"表明..." "意味着..." "反映了..."
"说明..." "体现了..." "预示着..."
"折射出..." "暴露了..." "突出了..."
```

### 论证连接词
```
逻辑递进: 首先...其次...最后
          基础是...进一步...深层是
因果关系: 因此...导致...结果是
对比: 相比之下...相较而言...反之亦然
补充: 此外...另外...值得注意的是
```

### 数据表述（精确但可读）
```
✅ "增长了10倍，从10亿美元升至100亿美元"
✅ "2024年前3月，XXX同比增长150%"
✅ "调查显示，78%的受访者..."
✅ "数据显示，这是10年来最高值"
❌ "增长很快" （太模糊）
❌ "相当可观" （太主观）
```

### 权衡表述（显得理性）
```
"一方面...另一方面..."
"固然...但同时..."
"虽然...不过..."
"既要...也要..."
示例: "虽然监管提高了市场准入门槛，但也为市场稳定性奠定了基础"
```

## 禁用表达

### 显得"廉价"的词汇
- "据说" "有人说" （来源不清）
- "觉得" "感觉" （太主观）
- "应该" "可能" （避免承诺）
- "有意思的是" "不得不说" （显得轻浮）

### 显得"学究"的词汇
- "鉴于此" "综合而言" "不容忽视" （过时）
- "笔者认为" "本文认为" （显得距离感）
- "需要指出的是" "必须说明的是" （显得冗余）

### 显得"不专业"的表述
- "让我们来看看" → "我们来看"
- "小编认为" → 完全删除，直接表述
- "听说" "好像" "似乎" → 改用确定表述或"据...表示"

## 常见改写示例

| 原文（过度简化） | 改写（Normal） |
|---|---|
| "这很重要" | "这意味着市场从无监管向有监管的转变，涉及资本流向、机构决策、投资策略调整" |
| "有人说..." | "根据XXX报告..." 或 "监管机构表示..." |
| "很多人认为" | "市场普遍预期..." 或 "数据显示..." |
| "可能会发生" | "按照目前趋势，预计会..." |
| "这个现象很复杂" | "这个现象涉及监管、资本、技术三个维度。首先...其次...最后..." |

## 质量检查清单

**开篇**:
- [ ] 问题清晰，无冗余
- [ ] 说明为什么重要（背景意义）
- [ ] 承诺本文分析角度
- [ ] 无感叹号、无修辞

**中间段落**:
- [ ] 每段包含 Statement + Proof + Application
- [ ] 论据至少2个，包含数据和案例
- [ ] 段落长度200-250字
- [ ] 使用权衡表述（一方面...另一方面）
- [ ] 避免绝对表述（必然、一定、肯定）

**结尾**:
- [ ] 总结核心观点（50-80字）
- [ ] 包含"隐含意义"（启示、前景）
- [ ] 不引入新信息
- [ ] 无转发激励、无emoji

**全文**:
- [ ] 字数符合目标（800-1200字）
- [ ] h2标题完全保留
- [ ] 所有{{image:}}、{{link:}}锚点保留
- [ ] 论据充分、逻辑严密
- [ ] 无"据说""觉得""好像"等模糊表述
