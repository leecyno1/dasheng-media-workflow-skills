# Hot版本（流量增强）语气规则

## 核心特征
- **目标**: 最大化传播力、话题性、转发率
- **受众心理**: 猎奇、对比、新鲜感
- **关键词**: 冲突、反转、突破、风险、机会

## 语气框架

### 开篇策略（前100字决定生死）

**必须包含以下1项**:

1. **冲突对比**
```
"你以为XXX？错了！" 
"曾经XXX，如今XXX"
"A说XXX，B说YYY，真相是..."
示例: "你以为稳定币没人要？错了！传统金融巨头一个月狂砸300亿"
```

2. **数据冲击**
```
"XXX增长了500%"
"99%的人不知道..."
"24小时内有XXX发生"
示例: "24小时内3家中央银行同时表态，加密市场发生了什么？"
```

3. **反常现象**
```
"本不该发生的事发生了"
"这违反了所有人的预期"
"黑天鹅正在降落"
示例: "日本自卫队混入极端分子，这信号比你想象更危险"
```

### 中间段落策略（维持热度）

**段落开头动词**（激发阅读欲望）:
- "暴露了..."
- "打破了..."
- "颠覆了..."
- "预示着..."
- "潜藏的...正在浮出水面"

**段落结构**（论据充沛）:
```
观点句（包含冲突词）
  ↓
论据1：数据/案例
论据2：权威观点
论据3：反面例子（为什么不这样）
  ↓
小结：这意味着什么
```

**长度建议**: 150-200字/段（保持阅读节奏感）

### 结尾策略（转发激励）

**必须包含以下1-2项**:

1. **问题激励**
```
"你的企业准备好了吗？"
"2024年，这会影响你的投资吗？"
"未来3个月内，XXX会发生什么？"
```

2. **转发激励**
```
"点'在看'，不错过任何破局机会"
"分享给你的投资圈朋友"
"这个观点你同意吗？评论区见"
```

3. **反思句**
```
"所有这些背后，隐藏的真相是..."
"未来的赢家，正在做这件事"
"3个月后回看这篇文，你就会明白"
```

## 词汇库

### 动词（行为感强）
```
入侵 破冰 入局 逆袭 翻盘 突破 暴雷 
内卷 打破 颠覆 瓦解 重塑 洗牌 逆转
潜伏 浮出 暴露 警告 预示 酝酿 蓄势
```

### 修饰词（冲击感强）
```
意想不到的 违反直觉的 出乎意料的 黑天鹅级的
系统性的 致命的 不可逆的 临界点的 关键时刻
隐藏的 潜在的 被忽视的 被压制的 真相级的
```

### 数据表述
```
❌ "增长了10%" → ✅ "增长了10倍"
❌ "下降了一些" → ✅ "暴跌50%，创历史新低"
❌ "许多人" → ✅ "99%的人"
❌ "相关研究表明" → ✅ "权威机构数据显示"
```

## 禁用表达

### 显得"软"的词汇
- "据悉", "有报道", "听说", "好像", "可能"
- "有一定的", "某种程度上", "相对而言"
- "值得关注", "有意思的是" （太温和）

### 显得"老气"的词汇
- "笔者认为", "需要指出的是", "鉴于此"
- "综合来看", "不容忽视"
- "让我们来探讨" （Web1时代表述）

### 容易翻车的词汇
- "必然", "一定", "肯定" （要为错误预测负责）
- "最终" （框死了对话空间）
- "只有...才" （过于绝对）

## 常见改写示例

| 原文（Normal） | 改写（Hot） |
|---|---|
| "稳定币监管框架已建立" | "稳定币牌照落地！传统金融巨头的"银行牌照时刻"" |
| "这可能影响市场" | "这触发了什么信号？99%的人还没反应过来" |
| "AI模型带来的风险" | "AI新模型引发华尔街紧急会议，金融业面临大洗牌" |
| "事件的背景是..." | "表面看是XXX，本质上是YYY——3年来最隐秘的渗透正在发生" |
| "未来值得观察" | "接下来3个月，这3大信号会决定你的投资方向" |

## 质量检查清单

- [ ] 开篇30秒内触发"冲突"/"数据"/"反常"中的1项
- [ ] 中间段落使用强动词（"暴露"/"颠覆"/"潜伏"等）
- [ ] 每段包含具体论据（数据、案例）
- [ ] 结尾包含问题激励或转发激励
- [ ] 避免使用"软"词汇（据悉、有报道等）
- [ ] 数据表述强劲（增长X倍，而非X%）
- [ ] 段落长度150-200字（保持节奏）
- [ ] 总体字数符合目标（1100-1500字）
